Methodology

Postman Value (BVA)

Formulas based on Forrester TEI, customer benchmarks, Postman State of API. See Phil's BVA Calculator for full methodology and sources.

Productivity: Devs x (Saved x 0.5) x 52 x Rate. Agent Mode: x 1.35
Governance: PRs x Mins/60 x ArchRate x 0.9
Shadow API Discovery: Undiscovered_APIs x Cost x 0.5
Outages: Outages x Cost x 0.3 x 0.5
Integration: Int x (TTFC - 0.25) x Rate
API Design: APIs x Hrs x 0.25 x Rate x 0.5 (Agent: 0.45)
Cost of Delay: Impact x Releases x Weeks x 0.50 (Agent: 0.60)
FTE Efficiency: QA_Engs x Salary x 65% x 50% (Agent: 80%)
Test Compression: Releases x Cycle_Hrs x 95% x Rate
Avoided Rework: Projects x 3 Devs x 3 Months x Rate

AI Effectiveness Index (AIEI)

From API Effectiveness as the Control Plane for AI (Khalsa, Nourian, Crawford, Gray).

AIEI = (wQ x Q) + (wL x L) + (wR x R) + (wC x C_norm)
Q: [1 - (Defect_Rate_AI / Defect_Rate_NonAI)] x Test_Pass_Delta x (1 - Rework_Rate_AI)
L: 1 - (Cycle_Time_AI / Cycle_Time_NonAI)
R: (1 - CFR_AI) x SLO_Attainment x Contract_Catch_Rate x Uptime_AI x Adaptability
C: (Labor_Saved - Overhead) / (Token + Infra + Tooling); C_norm = min(C_raw / Target_ROI, 1.0)
